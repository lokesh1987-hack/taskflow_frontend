# React + Vite

Setup Frontend (React App)

Open a new terminal and navigate to frontend folder:

cd frontend

Install dependencies:

npm install

Create .env file inside frontend/ folder:

VITE_API_URL=http://localhost:5000

Start frontend:

npm run dev

Frontend runs on:

http://localhost:5173
